## Installation

### Swift Package Manager

Add this to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/connect-auth-swift.git", from: "1.0.0")
],
targets: [
    .target(
        name: "YourApp",
        dependencies: ["ConnectAuth"]
    )
]
```
